# Student Performance Analysis Web Application

## Опис проєкту

Веб-застосунок призначений для аналізу успішності студентів університету. Система дозволяє переглядати оцінки, фільтрувати дані за факультетами та дисциплінами, а також візуалізувати результати у вигляді діаграм. Застосунок реалізовано з використанням Flask (backend), MySQL (база даних), HTML/CSS/JavaScript (frontend) та Chart.js (графіки).

## Основні функції

- Перегляд списку студентів, предметів та оцінок  
- Фільтрація даних за факультетом, групою та дисципліною  
- Побудова графіків успішності студентів  
- Додавання нових записів до бази даних  
- Виведення середнього балу та рейтингу  

## Технології

- **Backend:** Flask (Python)  
- **Database:** MySQL  
- **Frontend:** HTML, CSS, JavaScript  
- **Візуалізація:** Chart.js  

## Структура бази даних

База даних складається з чотирьох основних таблиць:

- `students` — інформація про студентів  
- `faculties` — перелік факультетів  
- `subjects` — навчальні дисципліни  
- `grades` — оцінки студентів
